Это MWSE-скрипт для Morrowind.

## **Что он делает:**
 - Моровые бури в игре могут заразить игрока моровыми болезнями, пока он в них находится. 
 - Сопротивления моровым болезням снижают вероятность заражения.
 - Закрытые шлемы (которые заменяют часть тела head) уменьшают базовый шанс заражения. 
 - Есть оповещение игрока о начале и окончании моровой бури. 
 - Настройки можно изменить из меню модов в игре, с мгновенным применением всех изменений. Есть возможность сброса к базовым настройкам.

## **Установка:**
Разместите файлы .lua в Morrowind\Data Files\mwse\mods\BlightStormInfection (создайте папку BlightStormInfection).

## **Зависимости:**
Наверняка для работы необходим MWSE и Morrowind.
